# Digital Timesheet System

Цифровой табель учёта рабочего времени

## Установка и запуск
```bash
npm install
npm run dev
```

Проект откроется на http://localhost:5173

## Production build
```bash
npm run build
npm run preview
```

## Технологии

- React 18
- TypeScript
- Vite
- SCSS
- date-fns

## Функционал

- Автоматическая генерация посещаемости
- Face ID real-time мониторинг
- Управление сотрудниками
- Фильтрация и поиск
- Статистика и аналитика
- LocalStorage persistence
